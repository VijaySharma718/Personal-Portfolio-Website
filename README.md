# Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, and achievements as a front-end developer.

## Live Demo
Check out the live website here: [Personal Portfolio Website](https://vijaysharma718.github.io/Personal-Portfolio-Website/)

## Technologies Used
- **HTML**: Structure of the website
- **CSS**: Styling and layout
- **JavaScript**: Interactivity and dynamic behavior

## Development Environment
- **IDE**: Visual Studio Code (VSCode)

## Features
- Clean and responsive design
- Interactive UI/UX
- Sections for about me, skills, projects, and contact information

## How to Use
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/vijaysharma718/Personal-Portfolio-Website.git
   ```
2. Open the project folder in VSCode or your preferred IDE.
3. Open `index.html` in your browser to view the website locally.
